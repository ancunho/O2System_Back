# 共同
共同返回格式，`data`里放数据。
```
{
  status: 0,   //（Numer）状态码【0表示成功，非0表示出错】
  msg: '成功',  //（String）信息
  data: any,    //（Any）数据
}
```

## 分页
```
【请求】
{
  username: 'admin',   //（String）账号
  password: 'password', //（String）密码
}

【响应】
{
  username: 'admin',   //（String）账号
  token: 'qeasdzxc', //（String）Token
}
```
